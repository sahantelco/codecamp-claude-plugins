# Codecamp Claude Plugins

Internal Claude Code plugins for Codecamp engineering teams.

## Plugins

| Plugin | Description |
|--------|-------------|
| [codecamp-i18n](plugins/codecamp-i18n/README.md) | i18n audit for Angular/Transloco projects — detects missing translations and unregistered keys |

## Install

```

```
